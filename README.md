# DATX11-VT24-13
DATX11-24-13_Python Tutorial Package

## Första körning av projektet
1. Installera Docker Desktop

3. Starta container
    ```javascript
    docker compose up -d
    ```

#### Backend:
1. Gå till backend directoriet

2. Kör följande kommandon i terminal:
    ```
     npm install
    ```
    ```
     npm run backend_setup
    ```
